# My-Data-Mining-Machine-Learning-Projects

A comprehensive repository for data mining and machine learning projects, now featuring advanced web scraping capabilities.

## 🆕 Web Scraping Setup

This repository now includes everything needed for **undetected web scraping** using SeleniumBase CDP mode:

### Quick Setup
```bash
# Install all dependencies
pip install -r requirements.txt

# Run automated setup
python setup_scraping.py

# Test with Nike scraping example
python run_examples.py raw_nike
```

### Features
- **🛡️ Bypass bot detection** - Uses SeleniumBase CDP mode for stealth
- **🤖 Handle CAPTCHAs** - Automatic CAPTCHA solving with PyAutoGUI
- **🌐 Support protected sites** - Works with Cloudflare, DataDome, Kasada, etc.
- **📊 Data mining integration** - Scraped data ready for ML analysis

### Web Scraping Examples
- `web_scraping/examples/raw_nike.py` - Nike product scraping
- `web_scraping/examples/cdp_basic_examples.py` - Basic CDP patterns
- More examples following undetected-testing patterns

See `web_scraping/README.md` for detailed documentation.
